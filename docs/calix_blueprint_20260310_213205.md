# MVP Scope

Core network device management, basic subscriber provisioning, real-time monitoring dashboard, simple billing integration,REST APIs, and a responsive web interface. Focus on single-tenant deployment with one broadband technology (fiber/GPON)to validate core value proposition.

# Competitive Landscape

The market is dominated by established players like Calix, ADTRAN, and Nokia, focusing on traditional OSS/BSS capabilities.Opportunities exist for cloud-native solutions with better UX, AI integration, and modern API-first architectures that enable fasterinnovation cycles.
